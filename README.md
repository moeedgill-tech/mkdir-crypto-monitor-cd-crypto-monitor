# Crypto Market Insights

A web application that fetches and visualizes real-time cryptocurrency market data.

## Features
* **Real-time Search**: Search for any cryptocurrency to view its latest 7-day price trend.
* **Interactive Charts**: Responsive line charts using Recharts for seamless data visualization.
* **Auto-completion**: A dynamic dropdown menu to easily find your favorite coins.

## Technologies Used
* **Next.js**: Framework for building the React application.
* **Tailwind CSS**: For a clean, modern UI design.
* **Recharts**: For rendering interactive financial charts.
* **CoinGecko API**: Providing live cryptocurrency market data.
